1. Shebang: `#!/usr/bin/env python3`
2. Docstring: "Evo-RL environment verification. Checks CUDA, key dependencies, and pretrained weight paths. Target: complete in under 10 seconds."
3. Imports: `sys`, `os`, `time` only at top level. All other imports inside try/except to catch missing packages.
4. Record start time with `time.time()`
5. **CUDA check:**
   - `import torch` inside try/except
   - `torch.cuda.is_available()` -> cuda_ok bool
   - If cuda_ok: get `torch.cuda.get_device_name(0)` and `torch.cuda.get_device_properties(0).total_memory / 1e9`
   - Print: `CUDA: OK | RTX 4090 (24.0 GB)` or `CUDA: NOT AVAILABLE`
6. **Dependency version checks** — for each of these packages, import inside try/except and print version:
   - `torch` (already imported) -> `torch.__version__`
   - `transformers` -> `transformers.__version__`
   - `accelerate` -> `accelerate.__version__`
   - `datasets` -> `datasets.__version__`
   - `lerobot` -> `lerobot.__version__`
   - `wandb` -> `wandb.__version__`
   - Format: `  torch: 2.7.1+cu126` (or `  torch: IMPORT FAILED` on exception)
   - Track failures in a list
7. **Weight path checks:**
   - Compute PRETRAINED as `os.path.join(os.path.dirname(os.path.abspath(__file__)), "..", "pretrained")`
   - Check 3 directories: `pi05_base`, `siglip-so400m-patch14-384`, `gemma-3-270m`
   - For each: `os.path.isdir(path)` -> print `  pi05_base: OK (/abs/path)` or `  pi05_base: MISSING (/abs/path)`
   - Track missing in a list
8. **Elapsed time:** print `Elapsed: {elapsed:.1f}s`
9. **Final status:**
   - If cuda_ok AND no import failures AND no missing weights: print `STATUS: READY` and `sys.exit(0)`
   - Otherwise: print `STATUS: NOT READY -- fix issues above` and `sys.exit(1)`

CRITICAL constraints:
- Do NOT instantiate any model (no `from_pretrained`, no `AutoModel`, no weight loading)
- Do NOT import heavy modules beyond the 6 listed above
- The script MUST complete in under 10 seconds on any machine with CUDA
- Make the file executable with `chmod +x`
