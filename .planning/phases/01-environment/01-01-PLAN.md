**Functions to implement:**

1. `is_lfs_pointer(path: Path) -> bool`
   - Open file in binary mode, read `len(LFS_MAGIC)` bytes
   - Return `f.read(len(LFS_MAGIC)) == LFS_MAGIC`
   - Catch `OSError`, `PermissionError` → return False

2. `find_sample_parquet(root: Path) -> Path | None`
   - Try `V30_DATA_GLOB` first, then `V21_DATA_GLOB`
   - Return first match from `list(root.glob(pattern))[:1]`
   - Return None if neither glob finds anything

3. `check_dataset(root: Path) -> dict`
   Returns a dict with keys: `name`, `path`, `info_ok`, `info`, `issues`, `warnings`, `stage_ready` (dict with keys "1", "2", "3"), `compatible`, `lfs_status`, `v30_file_naming`, `v21_file_naming`, `stats_present`.

   Perform these checks in order:

   a. **meta/info.json existence**: Try `json.load(open(root / "meta" / "info.json"))`. If missing, add blocker B-01 with fix: "Dataset directory may be wrong — expected meta/info.json at {path}". Set `info_ok=False`, return early.

   b. **Version check**: Parse `info["codebase_version"]`. Compare major version number:
      - Extract major with `int(ver.split(".")[0].lstrip("v"))` (no packaging import needed)
      - If major < 3: add blocker with `codebase_version` value and fix instructions
      - Use exact fix: `python -m lerobot.datasets.v30.convert_dataset_v21_to_v30 --repo-id=wzt/Kai0 --root=/media/wzt/cfy/pi-finetune/Kai0_dataset/Task_A --push-to-hub=false`
      - Note in fix: "Output will be written to `--root`/`--repo-id`, NOT in-place. Adjust `--repo-id` to desired output folder name."

   c. **LFS pointer check**: Call `find_sample_parquet(root)`. If sample found, call `is_lfs_pointer(sample)`. If is LFS pointer:
      - Add blocker: "Parquet files are Git LFS pointers (checked: {sample.relative_to(root)})"
      - Fix: `cd {root} && git lfs pull`
      - Note: "Fix B-LFS before running conversion. LFS pull must complete fully before conversion."

   d. **v3.0 file naming check**: Count `list(root.glob(V30_DATA_GLOB))` and `list(root.glob(V21_DATA_GLOB))`. If only v2.1 files found (v30_count == 0 and v21_count > 0): add blocker "File naming uses v2.1 convention (episode_XXXXXX.parquet). Stage 2 value_infer expects file-NNN.parquet". Fix: "Run full conversion (B-version fix above) which renames files."

   e. **Required fields check**: Check `info["features"]` for presence of `"action"` and `"observation.state"`. Check for at least one key starting with `"observation.images."`. Missing fields → blocker.

   f. **stats.json check**: Check if `(root / "meta" / "stats.json").exists()`. If missing: add WARNING (not blocker): "meta/stats.json missing — value_train normalizer will skip normalization (may cause unstable training). Run `python -m lerobot.scripts.compute_stats` after conversion."

   g. **Named fields check**: For `action` and `observation.state` in features, check if `features[field].get("names")` is not None. If null: add WARNING: "Feature '{field}' has no named joints — numeric slicing required for joint selection."

   h. **Stage readiness**: Stage 1, 2, and 3 are all BLOCKED if any blocking issue exists. They are all READY only if zero blockers. (All stages depend on the same baseline compatibility for this pipeline.)

   i. **overall compatible**: `compatible = len(issues) == 0`

4. `format_report(results: list[dict]) -> None`
   Print to stdout. Use Unicode symbols directly (✓, ⚠, ✗) — no termcolor dependency.

   Format per dataset:
   ```
   [{dataset_name}]
     Path: {path}
     Version: {version} ✓  (or ⚠  (expected v3.0))
     Robot: {robot_type}
     Episodes: {total_episodes:,} | Frames: {total_frames:,} | FPS: {fps}
     Action shape: [{shape}] | State shape: [{shape}]
     Cameras: {comma-separated camera names}
     Data files: Real parquet ✓  (or  GIT LFS POINTERS ⚠)
     Stage 1 (value-train): READY ✓  (or BLOCKED ✗)
     Stage 2 (value-infer): READY ✓  (or BLOCKED ✗)
     Stage 3 (policy-train): READY ✓  (or BLOCKED ✗)
     Overall: COMPATIBLE ✓  (or NEEDS CONVERSION ⚠)

     Blocking Issues:
     [B-01] {issue description}
            Fix: {exact command}
            {optional note line}
     ...

     Warnings:
     [W-01] {warning description}
   ```

   After all datasets, print summary:
   ```
   === Summary ===
   Compatible: {N}/{total} datasets
   Incompatible: {M}/{total} datasets
   ```

5. `main() -> int`
   - `argparse.ArgumentParser` with `--datasets nargs="+"` (default=DEFAULT_DATASETS)
   - For each path: call `check_dataset(Path(p))`
   - Call `format_report(results)`
   - Return `0` if all compatible, `1` if any have issues
   - `sys.exit(main())`
