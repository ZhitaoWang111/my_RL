       echo "Code target: ${A100_CODE_DIR}"

5. **Code sync function** (exact command from D-02 user decision in CONTEXT.md):
   ```
   sync_code() {
       echo "Syncing code to A100..."
       rsync -avz --progress \
           --exclude='.git' \
           --exclude='outputs/' \
           --exclude='pretrained/' \
           "${WORK_DIR}/" \
           "${A100_HOST}:${A100_CODE_DIR}/" \
           -e "${SSH_CMD}"
       echo "Code sync complete."
   }
   ```

6. **Weight sync function** (per D-08, D-10 from CONTEXT.md):
   ```
   sync_weights() {
       if [ ! -d "${WORK_DIR}/pretrained" ]; then
           echo "ERROR: No pretrained/ directory found. Run download_weights.sh first."
           exit 1
       fi
       echo "Syncing weights to A100..."
       rsync -avz --progress \
           "${WORK_DIR}/pretrained/" \
           "${A100_HOST}:${A100_PRETRAINED_DIR}/" \
           -e "${SSH_CMD}"
       echo "Weight sync complete."
   }
   ```

7. **Argument dispatch**: `case "${1:-}" in code) sync_code;; weights) sync_weights;; all) sync_code && sync_weights;; *) usage; exit 1;; esac`

8. **Make executable**: `chmod +x`
